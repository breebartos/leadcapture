LeadCapture
One-time setup — follow these steps to connect your Supabase database. You only do this once.
Step 1

Go to supabase.com and create a free account. Then click New Project and give it a name like leadcapture. Wait for it to finish provisioning (~1 min).

Step 2 — Create the database tables

In your Supabase project, go to SQL Editor and run this query:

-- Create leads table CREATE TABLE leads ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, name TEXT NOT NULL, company TEXT, title TEXT, notes TEXT, linkedin TEXT, email TEXT, phone TEXT, challenges TEXT, opportunities TEXT, photo_req BOOLEAN DEFAULT false, photo_cap BOOLEAN DEFAULT false, stage INTEGER DEFAULT 0, event TEXT, added_by TEXT, created_at TIMESTAMPTZ DEFAULT now(), updated_at TIMESTAMPTZ DEFAULT now() ); -- Create activity log table CREATE TABLE lead_log ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, lead_id UUID REFERENCES leads(id) ON DELETE CASCADE, text TEXT NOT NULL, author TEXT, created_at TIMESTAMPTZ DEFAULT now() ); -- Enable Row Level Security ALTER TABLE leads ENABLE ROW LEVEL SECURITY; ALTER TABLE lead_log ENABLE ROW LEVEL SECURITY; -- Allow authenticated users full access CREATE POLICY "Authenticated users can do everything" ON leads FOR ALL TO authenticated USING (true) WITH CHECK (true); CREATE POLICY "Authenticated users can do everything" ON lead_log FOR ALL TO authenticated USING (true) WITH CHECK (true);
Step 3 — Add your team's logins

In Supabase go to Authentication → Users → Invite user. Add an email and password for Bree and one for Pinaki. They'll use these to log in.

Step 4 — Paste your project keys below

In Supabase go to Project Settings → API. Copy the Project URL and the anon / public key.
